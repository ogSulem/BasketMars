# BasketballGame

Мини-игра «Баскетбол» для Android (Java 17, Gradle, OkHttp).

## Сборка и запуск
1. Откройте проект в Android Studio (Arctic Fox+).
2. Убедитесь, что установлен JDK 17 и Android SDK 34.
3. Синхронизируйте Gradle.
4. Соберите и запустите модуль `app` на устройстве или эмуляторе (`minSdk 21`).

## Игровые режимы
| Режим | Описание |
| --- | --- |
| **Аркада** | Бесконечная игра на рекорд, сложность растёт с очками. |
| **На время** | 60 секунд на набор максимального количества очков. |
| **Онлайн дуэль** | 90‑секундный матч 1 на 1 с «призрачным» соперником. Видите его траекторию и счёт в реальном времени. |

Выбор режима доступен из главного меню (кнопка «Играть» → диалог выбора).

## Управление
- Потяните мяч свайпом вверх — направление и скорость определяются жестом.
- Кнопка «Заново» появляется после броска и позволяет мгновенно повторить попытку.
- В режиме онлайн отображается таймер, счёт соперника и полупрозрачный мяч соперника.

## Сетевые возможности
- В `GameActivity` используется `OnlineMatchClient` (WebSocket, OkHttp 4.12.0).
- Текущий URL `wss://example.com/match` — заглушка, замените на боевой адрес сервера матчмейкинга.
- Клиент отправляет снепшоты (`x`, `y`, `moving`, `score`) ~каждые 60 мс и отрисовывает призрачный мяч соперника.

## Контент и ресурсы
- В `BallSelectActivity` доступны скины мячей, сеток и фоновых градиентов. Разблокировка идёт через достижения в `GameView`.
- Используются временные изображения `ball*.png`, `hoop*.png`. Замените их в `app/src/main/res/drawable/` на собственные ассеты.
